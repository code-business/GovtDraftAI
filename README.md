# GovDraft AI - Government Submission POC

A secure, AI-powered tool for drafting formal government letters in the healthcare and medical sector.

## Features
- **Department Specialization**: Tailored drafting for Oncology, Immunology, Virology, and more.
- **AI-Powered**: Integrates with Gemini 1.5 Flash for professional, formal letter generation.
- **Reference-Based**: Upload your rough notes or demo letters to guide the AI.
- **Modern UI**: Built with React 19, TypeScript, and Tailwind CSS v4.

## Setup
1. **API Key**: Add your Gemini API key to the `.env` file:
   ```env
   VITE_GEMINI_API_KEY=your_key_here
   ```
2. **Install Dependencies**:
   ```bash
   npm install
   ```
3. **Run Development Server**:
   ```bash
   npm run dev
   ```

## Tech Stack
- React 19
- Vite 8
- Tailwind CSS v4
- Lucide React (Icons)
- Google Generative AI SDK
